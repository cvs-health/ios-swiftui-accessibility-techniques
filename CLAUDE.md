# Project Rules

## Copyright Header (Required)

Every file in this project must include the Apache 2.0 copyright header for CVS Health. This is a legal requirement for this open source project.

### Rules

1. **New files**: Every new file must include the copyright header at the top. Use the current year only (e.g., `Copyright 2026 CVS Health`).
2. **Edited files**: When modifying an existing file, the copyright year must be updated to include the current year. If the file was created in a prior year, use a range (e.g., `Copyright 2024-2026 CVS Health`). If it was created this year, use just the current year (e.g., `Copyright 2026 CVS Health`).
3. **Never remove or omit** the copyright header from any file.
